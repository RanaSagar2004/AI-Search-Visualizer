# AI-Search-Visualizer
Python GUI tool to visualize BFS, DFS, and A*

Visualize pathfinding algorithms like:
- BFS
- DFS
- A* (Manhattan + Euclidean Heuristics)

Built using Python and Tkinter for students and developers learning AI.

## Features
- Interactive grid interface
- Adjustable speed and heuristics
- Clear visualization of search process
